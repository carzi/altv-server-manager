# altv-server-manager [ BETA ]

A tool to help you manage [alt:V Multiplayer](https://altv.mp/) server.

## How to use?

```yaml
altv init
    -nologo
    -win
    -linux
    -release
    -rc
    -dev
    -ts
    -js
    -csharp # soon
    -rust # soon

altv update
    -win
    -linux
    -release
    -rc
    -dev
    -ts
    -js
    -csharp # soon
    -rust # soon
    -server
    -data
    -module

altv dev
    -ts
    -js
    -csharp # soon
    -rust # soon

altv build
    -m
    -ts
    -csharp # soon
    -rust # soon
```
